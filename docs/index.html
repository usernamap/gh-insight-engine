🔐 Quick Start - Authentication Made Simple

🎯 Super Fast Setup (< 2 minutes)

# 1️⃣ Get your GitHub token
curl -X POST "https://api.gh-insight-engine.com/auth/login" \
     -H "Content-Type: application/json" \
     -d '{
       "username": "your_github_username",
       "fullName": "Your Full Name", 
       "githubToken": "ghp_your_classic_token_here"
     }'

# 2️⃣ Start collecting insights immediately!
curl -X POST "https://api.gh-insight-engine.com/users/your_username" \
     -H "Authorization: Bearer your_jwt_token"

🔑 GitHub Token Setup (Super Easy!)

Quick Setup Guide:

  1. 🎯 Go to GitHub → Settings > Tokens
  2. 🆕 Create new → Select "Classic Token"
  3. ✅ Check permissions:
    • repo → 🔒 Private repository access (complete analysis)
    • user → 👤 Profile information (detailed insights)
    • read:org → 🏢 Organization data (team analytics)
  4. 🎉 Copy token → Use in the request below!

🚨 Pro Tips for Success

  • 💡 Token expires? Set 90+ days for best experience
  • 🔒 Keep it secure - Never commit tokens to code
  • 🚀 Test immediately after generation
  • 🎯 Save the JWT for all future requests

🎉 What You Get Back

  • 🎫 Powerful JWT token (24h validity)
  • ✅ Permission verification
  • 🏆 Access level confirmation
  • ⚡ Instant API readiness
