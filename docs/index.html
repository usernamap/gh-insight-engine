GitHub Insight Engine API (0.1.4)

Download OpenAPI specification:

URL: https://github.com/org/gh-insight-engine License: MIT

API REST pour l'analyse en profondeur des données GitHub avec insights IA.

Cette API permet d'analyser les profils GitHub, repositories et de générer des insights quantitatifs et qualitatifs grâce à l'intelligence artificielle.

Documentation complète de l'API

Authentification

L'API utilise l'authentification JWT Bearer Token. Pour vous connecter, utilisez votre token GitHub Classic.

Rate Limiting

  • Global: 100 requêtes par 15 minutes par IP
  • Authentification: 5 tentatives par 15 minutes par IP
  • Analyses: 10 analyses par heure par IP

Workflow d'analyse

  1. Connexion avec token GitHub Classic
  2. Lancement d'analyse utilisateur (GitHub + Analytics)
  3. Génération d'insights IA
  4. Consultation des résultats
